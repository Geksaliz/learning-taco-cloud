# Taco cloud

This application was developed as part of a study of the book Spring Boot in Action.

## 🛠 Tech Stack
- Java 17
- Spring Boot
- PostgreSQL
- Kafka
- Gradle
- Docker (optional)

## ⚙️ Getting Started

1. Run [auth service](https://github.com/Geksaliz/spring-authorization-server)
2. Run docker-compose.yml
3. Run application: `./gradlew bootRun`

## 🧪 Run test

`./gradlew test`
